# LeetCode 14. Longest Common Prefix

## Постановка
Дано масив рядків `strs`. Потрібно знайти найдовший спільний префікс усіх рядків із масиву. Якщо спільного префіксу не існує — потрібно повернути порожній рядок `""`.

### Формат вхідних даних
- `strs` — масив рядків, 1 ≤ `strs.length` ≤ 200
- кожен рядок має довжину від 0 до 200 символів
- усі символи — латинські літери нижнього або верхнього регістру

### Формат результату
- Один рядок — спільний префікс усіх рядків або порожній рядок

### Приклади
1. `strs = ["flower","flow","flight"]` → `"fl"`
2. `strs = ["dog","racecar","car"]` → `""`

### Ідея розв'язку
- Узяти перший рядок за основу префікса
- Поступово скорочувати цей префікс, поки він не стане початком кожного наступного рядка
- Якщо префікс спорожнів — повернути `""`

Складність рішення: `O(n * m)`, де `n` — кількість рядків, `m` — довжина поточного префікса.
